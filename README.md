# Passo a Passo para Execução

Siga os comandos abaixo no terminal do VSCode para configurar e rodar a aplicação.

## 1. Clonar o repositório

```bash
git clone https://github.com/kaiquehsp/AV2-TPll.git
```

## 2. Entrar na pasta
```bash
cd AV2-TPll
```

## 3. Instalação das dependências
```bash
npm install
```

## 4. Instalação global do typescript
```bash
npm install -g typescript
```

## 5. Compilando o typescript

```bash
cd src/ts
npx tsc
```

## 6. Execução do Sistema 

Na raiz do projeto
```bash
node dist/app/app.js
```
